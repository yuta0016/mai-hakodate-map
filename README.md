# 菅野真衣のマイペース♡マイワールド「函館旅フォトブック〜まいきゃっち!!〜」

菅野真衣のマイペース♡マイワールド「函館旅フォトブック〜まいきゃっち!!〜」のロケ地を紹介するファンサイトです。

🔗 **サイトURL**: https://yuta0016.github.io/mai-hakodate-map/

## 📖 概要

このサイトは、菅野真衣のフォトブック『函館旅フォトブック〜まいきゃっち!!〜』の撮影スポットや、函館への観光情報を提供するファンメイドサイトです。

### 主なコンテンツ

- **撮影スポット情報**: 撮影スポットの詳細情報、スポットは随時追加予定
- **函館へのアクセス情報**: 東京、大阪、名古屋、福岡、札幌から函館への交通手段別ガイド
- **割引乗車券情報**: 函館市電・バスの1日・2日乗車券についての紹介
- **関連情報**: 物書きについてのページ、お問い合わせフォーム

## 🛠 技術スタック

- **HTML5** - マークアップ
- **CSS3** - スタイリング
- **Vanilla JavaScript** - インタラクティブ機能
- **Google Analytics** - アクセス解析
- **Google Maps API** - 地図表示

### フレームワーク・ライブラリ

- フレームワークなしの軽量設計
- フォント: Zen Maru Gothic

## 🚀 使用方法

1. リポジトリをクローン
```bash
git clone https://github.com/yuta0016/mai-hakodate-map.git
cd mai-hakodate-map
```

2. ローカルサーバーで実行
```bash
# Pythonを使用する場合
python -m http.server 8000

# またはお好みのローカルサーバーで実行
```

3. ブラウザで `http://localhost:8000` にアクセス
